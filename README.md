# criar

Protótipo clicável do wizard **/criar** da Proteína Lúdica — fluxo de auto-serviço onde um profissional constrói um assistente digital IA.

- **Live:** criar.proteinaludica.com
- **Estado:** protótipo de validação (não é produção). Dados não são guardados.
- **Stack:** HTML/CSS/JS num único ficheiro (`index.html`). Sem build, sem dependências.

## Deploy
Vercel, ligado a este repo. `vercel.json` faz cleanUrls e força `noindex` (protótipo).

## Nota
Wizard genérico — para qualquer profissão. Saúde é mercado de entrada, não o âmbito.
